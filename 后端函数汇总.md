### `parse_resume(pdf_file_path: str)` ⭐

**作用**：解析 PDF 简历文件，返回结构化分析结果

**输入**：
| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `pdf_file_path` | `str` | PDF 文件的完整路径 | `"C:/temp/resume_abc123.pdf"` |

**输出**：

```python
Dict[str, Any]  # 字典格式的结构化数据
```

**返回数据结构**：

```python
{
    "basic_info": {
        "name": "张三",
        "contact": "138xxxx",
        "years_of_experience": 5
    },
    "technical_skills": {
        "programming_languages": ["Python", "Java"],
        "frameworks": ["Django", "Spring"],
        "databases": ["MySQL", "Redis"],
        "tools": ["Git", "Docker"]
    },
    "work_experience": [
        {
            "company": "某某科技",
            "position": "高级工程师",
            "duration": "2020-至今",
            "responsibilities": ["负责...", "主导..."]
        }
    ],
    "projects": [
        {
            "name": "电商平台",
            "technologies": ["Python", "MySQL"],
            "highlights": ["高并发设计"],
            "contribution": "独立开发"
        }
    ],
    "assessment": {
        "technical_depth_score": 8,
        "technical_breadth_score": 7,
        "risk_points": ["跳槽频繁"]
    },
    "raw_text": "原始 PDF 文本内容..."
}
```

**调用示例**：

```python
# 1. 保存上传的 PDF 到临时文件
temp_pdf_path = TEMP_DIR / f"resume_{uuid4().hex}.pdf"
with open(temp_pdf_path, "wb") as f:
    f.write(uploaded_file.getvalue())

# 2. 调用 parse_resume 解析简历
analysis_result = parse_resume(str(temp_pdf_path))

# 3. 存储到 session state
st.session_state.resume_analysis = analysis_result
st.session_state.resume_file_name = uploaded_file.name
st.session_state.resume_uploaded = True

# 4. 删除临时文件
temp_pdf_path.unlink(missing_ok=True)
```

**异常处理**：

```python
try:
    analysis_result = parse_resume(str(temp_pdf_path))
except Exception as e:
    st.error(f"简历解析失败：{str(e)}")
```

**处理时间**：约 10秒（包含 PDF 提取 + AI 分析）

### 1.2 `format_resume_for_prompt(analysis: Dict[str, Any])`

**作用**：将简历分析结果格式化为文本，用于注入到面试官 AI 的 prompt 中

**输入**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `analysis` | `Dict[str, Any]` | `parse_resume()` 返回的分析结果 |

**输出**：

```python
str  # 格式化的简历摘要文本
```

**返回示例**：

```
【候选人基本信息】
姓名：张三
工作年限：5 年

【技术技能】
编程语言：Python, Java
框架：Django, Spring
数据库：MySQL, Redis

【工作经历】
1. 某某科技 - 高级工程师 (2020-至今)
   - 负责...
   - 主导...

【项目经验】
1. 电商平台
   技术栈：Python, MySQL
   亮点：高并发设计

【AI 评估】
技术深度：8/10
技术广度：7/10
风险点：跳槽频繁
```

**调用示例**：

```python
# 在调用 LLM 之前，注入简历信息
if st.session_state.resume_uploaded and st.session_state.resume_analysis:
    resume_info = format_resume_for_prompt(st.session_state.resume_analysis)

    augmented_system_prompt += (
        "\n\n【候选人简历信息】\n" + resume_info
        + "\n\n请根据候选人的背景，调整面试难度和问题方向，个性化面试。"
    )
```

**使用场景**：

- ✅ 面试开始时，将简历信息注入 system prompt
- ✅ AI 根据简历个性化提问

### 2.1 `ide_panel(key: str = "monaco_editor")` ⭐

**作用**：渲染完整的 IDE 面板（编辑器 + 提交按钮 + 历史记录）

**输入**：
| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| `key` | `str` | session_state 的键名 | `"monaco_editor"` |

**输出**：

```python
Dict or None
```

**返回值说明**：

- 如果用户提交了代码 → 返回字典：
  
  ```python
  {
      "code": "def two_sum(nums, target):\n    ...",
      "explanation": ""  # 不需要语音解释，为空字符串
  }
  ```
- 如果用户没有提交 → 返回 `None`

**界面布局**：

```
┌──────────────────────────────────────┐
│ 📝 代码编辑器              [❌ 关闭]  │
├──────────────────────────────────────┤
│                                      │
│  [代码编辑框]                        │
│                                      │
├──────────────────────────────────────┤
│ [📤 提交代码] [📋 历史] [🗑️ 清空]   │
├──────────────────────────────────────┤
│ （点击提交后）                       │
│ ✅ 代码已提交！请在对话区与 AI 讨论  │
└──────────────────────────────────────┘
```

**调用示例**：

```python
# 在 app.py 中，右侧 IDE 区域
if st.session_state.ide_visible:
    main_area, ide_area = st.columns([6, 4])

    with ide_area:
        # 调用 ide_panel 显示 IDE
        code_submission = ide_panel(key="current_code")

        # 处理提交结果
        if code_submission:
            # 将代码合并到用户输入（不需要语音解释）
            user_input = f"""
【我的代码】
```python
{code_submission['code']}
```

请点评这段代码。
"""

            # 添加到对话历史，发送给 AI
            st.session_state.history.append({
                "role": "user",
                "content": user_input
            })

```
**内部流程**：
```

1. 显示代码编辑器（调用 code_editor）
   ↓
2. 显示三个按钮（提交、历史、清空）
   ↓
3. 用户点击"提交代码"
   ↓
4. 直接提交代码（不需要录音）
   ↓
5. 保存提交记录（调用 save_code_submission）
   ↓
6. 返回 {code, explanation=""}
   
   ```
   
   ```

**Session State 管理**：

```python
# 相关状态
st.session_state.ide_visible          # IDE 是否显示
st.session_state.is_algorithm_mode    # 是否算法题模式
st.session_state.submitted_codes      # 历史提交记录列表
st.session_state.current_code         # 当前编辑器中的代码
```

**注意**：

- ✅ **不需要录音功能** - 录音是独立的语音面试模式
- ✅ 用户可以直接在对话区用文字或语音解释思路
- ✅ 代码提交后，AI 会自动点评

### 3.1 `transcribe_file(audio_file_path: str, api_key: str)`

**作用**：将音频文件转换为文字（ASR 语音识别）

**输入**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `audio_file_path` | `str` | 音频文件的完整路径 |
| `api_key` | `str` | 语音识别 API 密钥（从 config 导入） |

**输出**：

```python
str  # 识别的文字内容
```

**调用示例**：

```python
# 保存录音到临时文件
temp_wav = TEMP_DIR / f"audio_{uuid4().hex}.wav"
with open(temp_wav, "wb") as f:
    f.write(audio_value.getvalue())

# 调用 ASR 识别语音
try:
    text = asyncio.run(transcribe_file(str(temp_wav), STEPFUN_API_KEY))

    if text and text.strip():
        user_input = text.strip()
    else:
        st.warning("未识别到有效内容，请重试。")
except Exception as e:
    st.error(f"语音识别失败：{e}")
finally:
    temp_wav.unlink(missing_ok=True)
```

**处理时间**：约 2 秒

**异常处理**：

```python
try:
    text = asyncio.run(transcribe_file(str(temp_wav), STEPFUN_API_KEY))
except Exception as e:
    st.error(f"语音识别失败：{str(e)}")
```

### 5.1 `llm_stream_chat(history: List[Dict], user_input: str, system_prompt: str = None)` ⭐

**作用**：调用阿里云 LLM（qwen-max）进行流式对话

**输入**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `history` | `List[Dict]` | 对话历史列表 |
| `user_input` | `str` | 用户当前输入 |
| `system_prompt` | `str` | 系统提示词（可选） |

**输出**：

```python
Generator[str]  # 生成器，流式返回 AI 回复的文本片段
```

**调用示例**：

```python
# 流式调用 LLM
full_response = ""
for chunk in llm_stream_chat(
    st.session_state.history,
    user_input,
    system_prompt=augmented_system_prompt
):
    full_response += chunk
    # 实时更新 UI 显示
    reply_placeholder.markdown(full_response)

# 将 AI 回复添加到历史
st.session_state.history.append({
    "role": "assistant",
    "content": full_response
})
```

**对话历史格式**：

```python
st.session_state.history = [

### 6.1 `get_retrieved_context(query: str, domain: str, k: int, persist_dir: str)`

**作用**：从向量数据库中检索相关知识片段

**输入**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `query` | `str` | 检索查询文本 |
| `domain` | `str` | 知识领域（cpp/os/network/project） |
| `k` | `int` | 返回的片段数量 |
| `persist_dir` | `str` | 向量数据库目录 |

**输出**：

```python
str  # 检索到的知识片段文本
```

**调用示例**：

```python
# 在 LLM 调用前进行 RAG 检索
if st.session_state.enable_rag:
    persist_dir = str(Path(__file__).parent / "vector_db")

    try:
        retrieved = get_retrieved_context(
            user_input,
            domain=st.session_state.rag_domain,
            k=st.session_state.rag_top_k,
            persist_dir=persist_dir,
        )

        # 将检索结果注入到 system prompt
        if retrieved and retrieved.strip():
            augmented_system_prompt += (
                "\n\n参考知识库内容（仅供回答参考）：\n" + retrieved
            )

            # 记录 RAG 历史
            st.session_state.rag_history.append({
                "query": user_input,
                "retrieved": retrieved,
                "domain": st.session_state.rag_domain,
                "top_k": st.session_state.rag_top_k,
            })
    except Exception as e:
        st.warning(f"RAG 检索失败：{e}")
```

**使用场景**：

- ✅ 结合知识库内容进行面试
- ✅ 提供技术知识参考

### 7.1 `ai_report_stream(history: List[Dict], resume_analysis: Dict = None)` ⭐

**作用**：流式生成 AI 面试评价报告

**输入**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `history` | `List[Dict]` | 对话历史 |
| `resume_analysis` | `Dict` | 简历分析结果（可选） |

**输出**：

```python
Generator[str]  # 生成器，流式返回报告内容
```

**调用示例**：

```python
# 生成 AI 面试报告
if st.button("🤖 生成 AI 面试评价报告"):
    st.session_state.report_generating = True

    report_placeholder = st.empty()

    try:
        for partial_report in ai_report_stream(
            st.session_state.history,
            resume_analysis=st.session_state.get("resume_analysis")

            st.session_state.ai_report_text = partial_report
            report_placeholder.markdown(partial_report)
    except Exception as e:
        st.error(f"报告生成失败：{e}")

    st.session_state.report_generating = False
    st.rerun()
```
